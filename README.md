## Installation
The add-on is just a single file: **`NodeDungeonRouter.py`**

### 1) Download
- Go to the repo
- Open `NodeDungeonRouter.py`
- Download/save it somewhere (Desktop is fine)

### 2) Install in Blender
1. Open **Blender**
2. **Edit → Preferences → Add-ons**
3. Top-right dropdown (little **▼**) → **Install from Disk…**
4. Pick **`NodeDungeonRouter.py`**
5. Enable it (checkbox)

### 3) (Recommended) Save preferences
In the Preferences window, click Save Preferences so it stays enabled next time you open Blender.

## How to use
1. Open the **Shader Editor** or **Compositor** (not sure about Geomtery Nodes support)
2. Press **`N`** to open the sidebar (if it’s hidden)
3. Find the panel: **Node Dungeon Router** and click it to open the settings.
